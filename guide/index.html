          # 介绍

          vue-design是基于Vue2.0的仿照element-ui的自定义UI组件库，组件样式和api使用和element-ui保持一致，在设计组件库时，参考了element-ui、iview、antd、vant的设计、目录结构和编码思路。

          # npm安装

          npm i vue-design

          # 全局引入

          在入口文件main.js中写入以下内容s

          import Vue from "vue"
import vueDesign from "vue-design"
Vue.use(vueDesign)

          # 按需引入

          import {Button} from "vue-design"
Vue.use(Button)
或者
Vue.component(Button.name,Button)
